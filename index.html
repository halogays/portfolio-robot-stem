1) Pengenalan Proyek

Latar belakang masalah

Kebutuhan akan alat transportasi logistik otomatis yang bisa bergerak mandiri tanpa navigasi manual untuk meningkatkan efisiensi industri di masa depan.

Alat yang dibuat

  • Studuino Motherboard (Mikrokontroler)
  • ArTec Block Parts (Sasis & Body)
  • IR Photoreflector (Sensor Garis)

Fitur / manfaat

  • Deteksi jalur hitam otomatis dengan presisi tinggi.
  • Sistem bongkar pasang cepat (Modular) menggunakan ArTec Blocks.
  • Pemrograman visual yang mudah dipahami namun powerful.

Topik STEM terintegrasi

Science • Technology • Engineering • Mathematics

2) Tahap Perencanaan

Identifikasi masalah

  • Bagaimana cara sensor membedakan intensitas pantulan cahaya di lantai putih vs garis hitam?
  • Bagaimana mengatur kecepatan motor agar tidak keluar jalur saat tikungan?

Teori sains yang diterapkan

  • Optik: Hukum pemantulan cahaya infra merah.
  • Elektronika dasar: Konversi sinyal analog ke digital.

Alat & bahan

  • Main Unit Studuino, Sensor IR, DC Motor, Battery Box, Kabel USB, dan ArTec Blocks.
3) Desain & Prototyping

Sketsa robot

Menggunakan desain 2-wheel drive dengan satu ball caster di depan untuk mobilitas tinggi.

Arsitektur sistem (Input–Process–Output)

  • Input: IR Photoreflector (Mendeteksi garis).
  • Process: Studuino (Mengolah logika percabangan/Loop).
  • Output: DC Motor (Menggerakkan roda kanan/kiri).
4) Implementasi & Coding

Tahap perakitan

  1. Menyusun sasis menggunakan ArTec Blocks.
  2. Memasang Studuino dan menghubungkan sensor ke port A0/A1.
  3. Menyusun kode program menggunakan aplikasi Studuino (Icon Programming/Block Programming).

Flowchart

Start -> Cek Sensor -> Jika Hitam (Belok) -> Jika Putih (Lurus) -> Repeat.

5) Testing & Evaluasi

Skenario pengujian

  • Uji lintasan lurus 1 meter.
  • Uji belokan patah 90 derajat.
  • Uji respons sensor terhadap cahaya lampu ruangan.

Masalah yang ditemui

  • Robot sering keluar jalur karena sensor terlalu sensitif.
  • Roda sering selip di permukaan lantai yang licin.

Solusi

  • Kalibrasi ulang nilai Threshold pada software Studuino.
  • Menambahkan karet pada roda untuk meningkatkan grip.

Saran Pengembangan

  • Tambah lebih banyak sensor (3-5) agar gerakan lebih smooth.
  • Body lebih rigid, kabel lebih rapi (menggunakan ties).

Pendekatan STEM

Science

  • Penjelasan pantulan cahaya IR pada permukaan hitam (serap) dan putih (pantul).
  • Prinsip kerja Photoreflector dalam menangkap panjang gelombang cahaya.

Technology

  • Software: Studuino Programming Environment.
  • Hardware: Sensor ArTec, Motor DC, dan Studuino Board.
  • Debugging menggunakan log data sensor secara real-time.

Engineering

  • Alur: Perencanaan -> Desain ArTec Blocks -> Prototipe -> Testing.
  • Optimalisasi penempatan sensor agar dekat dengan permukaan lintasan.

Mathematics

  • Perhitungan Error Relatif IR Photoreflector.
  • $$Error Relatif(\%) = \left| \frac{x_{aktual} - x_{ref}}{x_{ref}} \right| \times 100\%$$
