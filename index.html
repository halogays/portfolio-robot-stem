I. Portfolio Showcase

1. Pengenalan Proyek

Smart Distance Tracker adalah robot yang dapat mendeteksi objek di depan. Ketika objek terlalu dekat, robot akan berhenti otomatis dan menyalakan LED + buzzer sebagai peringatan untuk mengurangi risiko benturan.

  • Alat: Robot berbasis Studuino + sensor IR.
  • Fitur utama: berhenti otomatis + alarm.
  • Manfaat: meningkatkan keamanan gerak robot/alat.

2. Tahap Perencanaan

  • Identifikasi masalah: potensi tabrakan robot.
  • Tujuan: robot berhenti otomatis.
  • Teori sains: pengukuran & ketidakpastian.
  • Komponen: Studuino, IR sensor, LED, buzzer, motor, baterai.

3. Desain & Prototyping

Kami membuat sketsa awal untuk menentukan posisi motor dan sensor. Sistem mengikuti prinsip Input – Process – Output.

4. Implementasi & Coding

Program dibuat menggunakan aplikasi Studuino (block coding). Logika utama:

  • Jika sensor > 7 → alarm ON + motor STOP
  • Jika sensor ≤ 7 → motor jalan

5. Testing & Evaluasi

  • Uji jarak (5–15 cm).
  • Uji cahaya terang–redup.
  • Uji konsistensi (pengukuran berulang).

Masalah

  • Sensor kurang stabil di cahaya terang.
  • Motor kadang terlambat berhenti.

Solusi

  • Kalibrasi threshold.
  • Perbaikan posisi sensor.
  • Program dioptimalkan (hapus delay tidak perlu).

6. Tim & Refleksi

Tim: Ferdinandus, Leonni, Keyko, Valentino, Christian. Kami belajar bahwa wiring rapi, sensor stabil, dan debug teratur sangat menentukan keberhasilan robot.

2. Technology (Studuino & ArTec Robo)

A. Pengenalan Studuino

Studuino adalah mikrokontroler pendidikan yang memudahkan perakitan dan pemrograman robot. Studuino punya port input (misalnya A0–A5) untuk sensor, dan port motor (misalnya M1–M2) untuk mengendalikan motor. Pemrograman dilakukan lewat aplikasi Studuino dengan block coding.

  • Peran Studuino di proyek ini:
    • Membaca IR Photoreflector (A1).
    • Menjalankan logika if–else.
    • Mengendalikan LED (A0), buzzer (A2), motor (M1).
  • Alasan memakai Studuino: mudah diprogram, cocok untuk pembelajaran robotika, dan kompatibel dengan ArTec.

B. Pengenalan ArTec Robo / ArTec Blocks

ArTec Robo adalah kit robotika yang memakai ArTec Blocks sebagai rangka robot. Blocks ini modular (bisa bongkar pasang), jadi desain bisa cepat diubah ketika evaluasi. ArTec membantu membuat rangka stabil dan posisi sensor lebih presisi.

  • Mudah dirakit dan dimodifikasi.
  • Struktur stabil.
  • Posisi sensor dapat diatur (tinggi/sudut).

C. Komponen

  • IR Photoreflector – A1
  • LED – A0
  • Buzzer – A2
  • Motor – M1
  • Baterai

D. Cara Kerja

  • Sensor > 7 → alarm ON & motor STOP
  • Sensor ≤ 7 → motor jalan

4. Mathematics (Langkah Hitung Rata-rata, Ketidakpastian, dan Error Relatif)

Di bagian matematika, kami mengolah data pengukuran berulang supaya hasilnya tidak cuma “kira-kira”. Jadi kami hitung rata-rata, lalu hitung ketidakpastian pengukuran berulang ($\Delta x$), dan terakhir hitung error/ketidakpastian relatif dalam persen.

A. Data yang dipakai

Data pengukuran berulang (cm): 3.0, 2.8, 2.6, 3.0, 2.8

B. Langkah 1 — Hitung rata-rata ($\bar{x}$)

Rumus rata-rata:
$\bar{x}=\frac{\sum x}{N}$

  • Jumlah data ($N$) = 5
  • $\sum x = 3.0+2.8+2.6+3.0+2.8 = 14.2$
  • $\bar{x} = \frac{14.2}{5} = 2.84\ \text{cm}$

C. Langkah 2 — Hitung $\sum x^2$ (dibutuhkan untuk rumus $\Delta x$)

Kita kuadratkan tiap data lalu dijumlahkan:

  • $3.0^2=9.00$
  • $2.8^2=7.84$
  • $2.6^2=6.76$
  • $3.0^2=9.00$
  • $2.8^2=7.84$
  • Jadi $\sum x^2 = 9.00+7.84+6.76+9.00+7.84 = 40.44$

D. Langkah 3 — Hitung ketidakpastian pengukuran berulang ($\Delta x$)

Rumus ketidakpastian pengukuran berulang (sesuai LKS):
$\Delta x=\frac{1}{N}\sqrt{\frac{N\sum x^2-(\sum x)^2}{N-1}}$

Substitusi angka:

  • $N\sum x^2 = 5 \times 40.44 = 202.2$
  • $(\sum x)^2 = (14.2)^2 = 201.64$
  • $N\sum x^2-(\sum x)^2 = 202.2 - 201.64 = 0.56$
  • $\frac{0.56}{N-1} = \frac{0.56}{4} = 0.14$
  • $\sqrt{0.14} \approx 0.374$
  • $\Delta x = \frac{1}{5}\times 0.374 \approx 0.0748\ \text{cm}$

E. Langkah 4 — Hitung error/ketidakpastian relatif (%)

Rumus ketidakpastian relatif:
$\text{Relatif}(\%)=\frac{\Delta x}{\bar{x}}\times 100\%$

  • $\frac{\Delta x}{\bar{x}} = \frac{0.0748}{2.84} \approx 0.0263$
  • Relatif $\approx 0.0263 \times 100\% = 2.63\%$

Kesimpulan (dari hitungan)

  • Rata-rata = 2.84 cm
  • Ketidakpastian = 0.0748 cm
  • Ketidakpastian/error relatif = 2.63%
