I. Portfolio Showcase

1. Pengenalan Proyek

Smart Distance Tracker adalah robot yang dapat mendeteksi objek di depan. Ketika objek terlalu dekat, robot akan berhenti otomatis dan menyalakan LED + buzzer sebagai peringatan untuk mengurangi risiko benturan.

  • Alat yang dibuat: Robot berbasis Studuino + sensor IR Photoreflector.
  • Fitur utama: berhenti otomatis + alarm ketika objek dekat.
  • Manfaat: simulasi sistem keamanan alat/transportasi otomatis.
  • Topik STEM: Science – Technology – Engineering – Mathematics.

2. Tahap Perencanaan

  • Identifikasi masalah: risiko alat bergerak menabrak objek.
  • Teori sains: jarak, intensitas IR, ketidakpastian pengukuran.
  • Komponen: Studuino, sensor IR, LED, buzzer, motor DC, driver, baterai, ArTec Blocks.
  • Dokumentasi: catatan fisika (LKS) & perencanaan komponen.

3. Desain & Prototyping

Kami membuat sketsa untuk menentukan posisi sensor, rute kabel, dan tata letak komponen. Kemudian robot dirakit menggunakan ArTec Blocks agar stabil.

  • Input: IR Photoreflector (A1)
  • Process: Studuino membaca & membandingkan nilai sensor
  • Output: Motor berhenti + alarm menyala

4. Implementasi & Coding

Program dibuat dalam bentuk blok pada Studuino: jika sensor membaca nilai lebih dari threshold, buzzer & LED menyala, motor berhenti.

  • Threshold: IR Photoreflector (A1) > 7
  • Output: buzzer ON, LED ON, motor brake
  • Else: buzzer OFF, LED OFF, motor jalan

5. Testing & Evaluasi

Skenario pengujian:

  • Uji sensor pada jarak berbeda menggunakan mistar
  • Uji respon motor ketika objek mendekat
  • Uji pada cahaya terang/redup
  • Uji konsistensi (pengukuran berulang) → hitung error/ketidakpastian

Masalah

  • Pembacaan sensor tidak stabil pada cahaya terang
  • Motor terlambat berhenti
  • Program awal kurang rapi (delay tidak perlu)

Solusi

  • Kalibrasi threshold
  • Perbaikan posisi sensor
  • Optimasi kode & hapus delay

1. Science (Besaran & Pengukuran)

Kami mempelajari pengukuran jarak menggunakan IR Photoreflector. Untuk memastikan data valid, dilakukan pengukuran berulang.

Kesalahan pengukuran

  • Kesalahan alat (skala, sensor)
  • Kesalahan manusia (pembacaan)
  • Kondisi percobaan (cahaya, getaran)
  • Teknik kurang sempurna (posisi sensor tidak konsisten)

Rumus

  • $\Delta x=\frac{1}{2}\times\text{skala terkecil}$
  • $\Delta x=\frac{1}{N}\sqrt{\frac{N\sum x^2-(\sum x)^2}{N-1}}$
  • Error relatif: $\frac{\Delta x}{\bar{x}}\times100\%$

Data pengukuran berulang

3.0, 2.8, 2.6, 3.0, 2.8

  • Rata-rata: 2.84 cm
  • Δx ≈ 0.0748 cm
  • Error relatif ≈ 2.63%
